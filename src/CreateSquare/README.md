# CreateSquare

`Strings`

[View on Edabit](https://edabit.com/challenge/zk8Xrof2x55u9FBJ5)

Create a function that takes a number as an argument and return the a string like square.

### Examples

```js
createSquare(-1) ➞ ""

createSquare(0) ➞ ""

createSquare(1) ➞ "#"

createSquare(2) ➞ "##\n##"

createSquare(3) ➞ "###\n# #\n###"

createSquare(4) ➞ "####\n#  #\n#  #\n####"
```

```js
"####
#  #
#  #
####"
```

### Notes

Square width <= 16383
