# Count Ones in a 2D Array

`Arrays`

[View on Edabit](https://edabit.com/challenge/ve7mQnJsjtFep97fm)

Create a function to count the number of `1`s in a 2D array.

### Examples

```js
countOnes([
  [1, 0],
  [0, 0]
]) ➞ 1

countOnes([
  [1, 1, 1],
  [0, 0, 1],
  [1, 1, 1]
]) ➞ 7

countOnes([
  [1, 2, 3],
  [0, 2, 1],
  [5, 7, 33]
]) ➞ 2
```
