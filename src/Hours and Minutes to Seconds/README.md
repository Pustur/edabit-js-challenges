# Hours and Minutes to Seconds

`Language Fundamentals` `Math`

[View on Edabit](https://edabit.com/challenge/JesaFi5ntBEbGT8bu)

Write a function that takes two integers `hours` `minutes` and converts them to seconds.

### Examples

```js
convert(1, 3) ➞ 3780

convert(2, 0) ➞ 7200

convert(0, 0) ➞ 0
```
