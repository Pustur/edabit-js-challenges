# Calculate Depth of Array

`Arrays` `Recursion`

[View on Edabit](https://edabit.com/challenge/JJtafXGmNegpQMp6p)

Given an array, write a function to calculate it's depth. Assume that a normal array has a depth of 1.

### Examplses

```js
depth([1, 2, 3, 4]) ➞ 1

depth([1, [2, 3, 4]]) ➞ 2

depth([1, [2, [3, 4]]]) ➞ 3

depth([1, [2, [3, [4]]]]) ➞ 4
```
